用Macbook執行遊戲可能會有在特效當掉或是無法開啟遊戲的風險
程式與相關所需音源、圖片等皆在「程式」資料夾中
執行方式：
(一)爬蟲
1.若「程式」資料夾中沒有library.txt的話，需先執行crawling.py，執行時有tqdm顯示進度，共有四個科目要執行
2.若要改變單字庫大小，請至crawling.py中Ln9修改am

(二)主程式
1.直接執行即可
2.若「程式」資料夾中沒有game_data.txt的話，會從等級1，0分開始；但顯示等級1，0分也有可能是寄過去的資料夾中目前儲存資料為等級1，0分。
3.遊戲資料要到死亡畫面才會自動儲存
4.從guideline回到主畫面時，若要進入遊戲，相對應鍵盤字母須按2次
5.某些電腦執行時可能出現鍵盤音效與爆炸音效未出現的狀況
